# Ejemplo de ChatGPT desde Python

https://openai.com/api/


```bash

mkdir ChatGPTPython
cd ChatGPTPython
touch chatgpt.py

code .

# Crear el venv
python3 -m venv venv   

# Activar el venv
source venv/bin/activate

pip install -r requirements.txt 

python3 -u chatgpt.py

# Desactivar el venv
deactivate

```
